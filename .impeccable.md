## Design Context

### Users
Peter Hartree — solo user on his MacBook Pro. A consultant/coach who does deep focused work and needs gentle-but-persistent reminders to take breaks. The app interrupts flow state, so the interruption itself needs to feel welcome, not hostile.

### Brand Personality
Warm, playful, companionable. The doge mascot is a friendly presence, not decoration. So Break should feel like a thoughtful friend tapping your shoulder — never a corporate wellness popup or a bland system alert.

### Aesthetic Direction
- **Palette**: AI Wow family — warm cream paper (#FFF9E3), dark ink, vivid accent colours (pink, orange, cyan, purple). The warmth is the identity.
- **Theme**: Light. The warm paper tone is the foundation. This is a daytime productivity companion.
- **Tone**: Lighthearted but not childish. The escalating messages already nail this — the design should match that voice.
- **Anti-references**: Generic macOS NSAlert dialogs. Corporate "employee wellbeing" tools. Stock-photo wellness apps.
- **Platform**: Native macOS (Swift/SwiftUI). Overlay windows at screensaver level. Not a web app.

### Design Principles
1. **Crafted, not templated** — Every element should feel intentionally designed. No default system styling left visible.
2. **Warm interruption** — The popup should make you smile, not groan. The doge earns its screen time.
3. **Clear hierarchy** — One message, one or two actions. Don't make the user think during a break prompt.
4. **Personality through restraint** — The copy does the heavy lifting for personality. The visual design supports it with warmth and polish, not gimmicks.
5. **macOS-native quality** — Should feel like it belongs on macOS. Smooth, fast, respects system conventions where it matters (window behaviour, keyboard shortcuts).
